Tu teléfono como cámara de seguridad

Convertí cualquier celular viejo en una cámara IP. Multi-cámara, grabación, detección de movimiento, intercom, control remoto — todo peer-to-peer con WebRTC.

Importante: esta página debe estar en HTTPS para que el navegador permita usar la cámara. Hosteala gratis en Netlify Drop, GitHub Pages, Vercel.
